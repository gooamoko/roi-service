Инструкция по запуску приложения под Linux (Тестировалось на Debian 11)

Все команды выполняются от имени привилегированного пользователя. Чтобы стать на время привилегированным пользователем,
можно выполнить команду
  sudo su
и ввести пароль пользователя.

1. Создайте пользователя и группу от имени которых будет работать приложение.
1.1 Создадим группу
  groupadd roi

1.2 Создадим пользователя
  useradd -g roi -d /opt/roi -s /usr/sbin/nologin roi

2. Распакуем файлы из архива (roi-service<version>.zip) в папку /opt/roi
Данную операцию проще всего выполнить в midnight commander, установить который можно командой
  apt-get install mc
После распаковки, в случае необходимости можно отредактировать конфигурационные файлы application.properties и logback.xml

3. Сменим пользователя-владельца у каталога /opt/roi командой
  chown -R roi:roi /opt/roi

4. Скопируем (либо перенесем) файл roi.service в папку /etc/systemd/system
Эту операцию так же удобнее всего выполнить в midnight commander

5. Обновим конфигурацию systemd командой
  systemctl daemon-reload

6. Запустим приложение roi командой
  systemctl start roi

7. Проверим статус запущенного приложения командой
  systemctl status roi

8. Включим автозапуск при запуске системы для приложения roi командой
  systemctl enable roi

9. Если нужно остановить приложение, то сделать это можно командой
  systemctl stop roi
